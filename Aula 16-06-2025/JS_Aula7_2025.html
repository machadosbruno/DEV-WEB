1. LocalStorage

Podemos armazenar um dado no navegador do usuário para usar mais tarde.

Para armazenar dados use o método setItem(). 

localStorage.setItem('nome', 'Fulano de Tal');

Acesse em inspecionar: Application >> Local storage >> file:// para ver o item armazenado.

Para pegar um item armazenado use getItem() 

console.log(localStorage.getItem('nome'));

Para remover um item armazendo use removeItem(): 

console.log(localStorage.removeItem('nome'));

Se um item não está mais armazenado o retorno de getItem() será null.

2. Transformando em JSON

Podemos transformar um dado em formato JSON: 

let nome = 'Fulano';
let nomeJSON = JSON.stringify(nome);

3. Retornando o valor

Para retornar o valor de JSON usamos: 

let retorno = JSON.parse(nomeJSON);

4. Vetores em localStorage

Aqui temos um vetor: 

var x = [2,3,4];

Se armazenarmos um vetor em localStorage, teremos os elementos retornados em uma String: 

localStorage.setItem('vetor', x);
let retorno = localStorage.getItem('vetor');
console.log(retorno);
console.log(typeof(retorno));

Para armazenar o vetor para usar mais tarde podemos transformá-lo em JSON: 

localStorage.setItem('vetor', JSON.stringify(x));
let retornoJSON = localStorage.getItem('vetor');
console.log(retornoJSON);
console.log(typeof(retornoJSON));

Para usar o retorno basta transformarmos em um vetor usando JSON.parse: 

let retorno = JSON.parse(localStorage.getItem('vetor'));
console.log(retorno);
console.log(typeof(retorno));

5. Temporizador

Existem duas opções de temporizador em JavaScript:

  1. setTimeout.
  2. setInterval.

5.1. setTimeout()

Para ajustar uma função para executar uma vez após 2 segundos usamos setTimeout(): 

let x = 2;
let saida;

window.onload = inicio;

function inicio() {
    saida = document.getElementById('saida');
    setTimeout(multi, 2000);
}

function multi() {
  x = x * x;
  saida.innerHTML = x;
}

Assim, setTimeout(funcao, tempo) aguarda a passagem do tempo e executa a funcao apenas uma vez.

5.3. setInterval()

Para ajustar um comando para executar a cada 2 segundos usamos setInterval(): 

let x = 1;
let saida;

window.onload = inicio;

function inicio() {
    saida = document.getElementById('saida');
    setInterval(soma, 2000);
}

function soma() {
  x = x + x;
  saida.innerHTML = x;
}

Assim, setInterval(funcao, tempo) aguarda a passagem do tempo e executa a funcao e repete indefinidamente o processo até que a página seja encerrada.
